# Steam Download Monitor

## Назначение
Python-скрипт для мониторинга скорости загрузки игр в Steam. Автоматически определяет установку клиента, парсит логи загрузок и выводит информацию в консоль.

## Ключевые возможности
- **Автоопределение пути Steam** (реестр Windows / стандартные пути Linux/macOS)
- **Парсинг логов** `content_log.txt` в реальном времени
- **Мониторинг с интервалом 1 минуту** в течение 5 минут
- **Отслеживание статуса**: загрузка, пауза, завершение
- **Вывод скорости загрузки** в МБ/с с идентификацией игры

## Основные функции
- `get_steam_path()` – находит путь к Steam через реестр или стандартные пути
- `parse_logs()` – анализирует логи, извлекает данные о загрузке
- `main()` – основной цикл мониторинга (5 итераций по 60 секунд)
- `run_in_background()` – настройка автозапуска для Windows

## Выходные данные
```
[14:30:00] Минута 1/5
Игра: Cyberpunk 2077 (AppID: 1091500)
Статус: Загружается (42%)
Скорость: 35.7 МБ/с
```

## Использование
```bash
# Стандартный запуск (5 минут мониторинга)
python steam_monitor.py

# Фоновый режим (Windows, добавление в автозагрузку)
python steam_monitor.py --background
```

## Технические особенности
- Работает на Windows, Linux, macOS
- Определяет скорость с конвертацией из KB/GB в MB
- Использует регулярные выражения для парсинга логов
- Обрабатывает отсутствие Steam или логов

## Требования
- Python 3.7+
- Установленный клиент Steam (для реального использования)
- Права на чтение файлов/реестра

## Формат логов Steam
Скрипт анализирует `Steam/logs/content_log.txt`, ищет:
- AppID игры (например: `AppID 1091500`)
- Процент загрузки (`Downloading 42%`)
- Скорость (`24.5 MB/sec`)
- Статус (`paused`, `Downloaded`)
